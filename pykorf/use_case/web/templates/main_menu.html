How to Use
  1. Apply Data (PMS / HMB) to populate pipe specs and fluid data.
  2. Use Global Settings to set sizing criteria and dummy pipes.
  3. Generate a Report or export to Excel for review.
  4. Click Save to write changes back to the .kdf.

Operations

Global Settings

Apply bulk modifications to all pipes — dummy pipes, dP/dL margin, rename, sizing criteria.
Apply Data

Apply PMS (pipe specs) and HMB (stream data) from Excel files.
Bulk Copy Fluids

Copy fluid properties from one pipe to multiple others. Supports include/exclude modes.
Reports

Generate reports, export to Excel, or import edited parameters back.
Model Info

View element statistics and a full sortable pipe list.
References

Design basis notes, SharePoint links and reference documents. Shortcuts auto-created next to the KDF.

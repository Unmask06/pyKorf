When to use this

OneDrive sometimes syncs a subfolder of a SharePoint library but only records the library root URL in Windows. This causes the Browse button to generate an incorrect SharePoint URL missing the intermediate folders.

How to find the correct URL:

  1. Open the folder in SharePoint via your browser.
  2. Copy the URL — it contains an id= parameter.
  3. URL-decode the id value (replace %2F→/, etc.).
  4. Prefix with https://tenant.sharepoint.com to form the direct URL.

Example:

Local folder:

C:\Users\UserName\CC7\25002 TAZIZ SALT PVC - ADNOC UAE - 2A-PROCESS

SharePoint URL:

https://cc7ges.sharepoint.com/sites/25002TAZIZSALT-ADNOCUAE/Shared Documents/2 EXECUTION/2. ENG/2A-PROCESS

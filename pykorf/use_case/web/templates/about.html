About pyKorf
v{{ version }} Released {{ release_date }}

pyKorf is a lightweight toolkit for engineers working with KORF hydraulic model files (.kdf).

KORF is powerful but its native interface offers limited bulk-editing and reporting capabilities. pyKorf bridges that gap — load a model, apply consistent pipe sizing criteria, import PMS / HMB data from Excel, and export a clean summary report, all from a browser.

Key Features
  • Read and write .kdf model files without opening KORF
  • Assign pipe sizing criteria (dP, velocity) from a built-in lookup table
  • Bulk-import fluid data from PMS and HMB spreadsheets
  • Export a pipe summary report to Excel
  • Apply project level parameters to entire model in one click
